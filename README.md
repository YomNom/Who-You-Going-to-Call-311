# Who-You-Going-to-Call-311-
Pothole on your street?  Missed trash pick-up?   Loud party next door?  Call 3-1-1.   The city of Cincinnati has put together a data portal where you can access a variety of data about the city (all data here: https://data.cincinnati-oh.gov/.).  This is an interactive visualization to help people understand 311 (Non-emergency) calls in 2022.
